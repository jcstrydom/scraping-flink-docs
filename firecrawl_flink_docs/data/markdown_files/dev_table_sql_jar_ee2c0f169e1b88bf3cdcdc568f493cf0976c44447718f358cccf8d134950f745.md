# JAR Statements  [\#](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/dev/table/sql/jar/\#jar-statements)

JAR statements are used to add user jars into the classpath or remove user jars from the classpath
or show added jars in the classpath in the runtime.

Flink SQL supports the following JAR statements for now:

- ADD JAR
- SHOW JARS
- REMOVE JAR

## Run a JAR statement  [\#](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/dev/table/sql/jar/\#run-a-jar-statement)

SQL CLI

The following examples show how to run `JAR` statements in [SQL CLI](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/dev/table/sqlclient/).

SQL CLI

```sql
Flink SQL> ADD JAR '/path/hello.jar';
[INFO] Execute statement succeeded.

Flink SQL> ADD JAR 'hdfs:///udf/common-udf.jar';
[INFO] Execute statement succeeded.

Flink SQL> SHOW JARS;
+----------------------------+
|                       jars |
+----------------------------+
|            /path/hello.jar |
| hdfs:///udf/common-udf.jar |
+----------------------------+

Flink SQL> REMOVE JAR '/path/hello.jar';
[INFO] The specified jar is removed from session classloader.
```

## ADD JAR  [\#](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/dev/table/sql/jar/\#add-jar)

```sql
ADD JAR '<path_to_filename>.jar'
```

Add a JAR file to the list of resources, it supports adding the jar locates in a local or remote [file system](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/deployment/filesystems/overview/). The added JAR file can be listed using [`SHOW JARS`](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/dev/table/sql/jar/#show-jars) statements.

### Limitation  [\#](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/dev/table/sql/jar/\#limitation)

Please don’t use `ADD JAR` statements to load Hive source/sink/function/catalog. This is a known limitation of Hive connector and will be fixed in the future version. Currently, it’s recommended to follow this [instruction](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/connectors/table/hive/overview/#dependencies) to setup Hive integration.

## SHOW JARS  [\#](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/dev/table/sql/jar/\#show-jars)

```sql
SHOW JARS
```

Show all added jars which are added by [`ADD JAR`](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/dev/table/sql/jar/#add-jar) statements.

## REMOVE JAR  [\#](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/dev/table/sql/jar/\#remove-jar)

```sql
REMOVE JAR '<path_to_filename>.jar'
```

Remove the specified jar that is added by the [`ADD JAR`](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/dev/table/sql/jar/#add-jar) statements.

Attention REMOVE JAR statements only work in [SQL CLI](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/dev/table/sqlclient/).
