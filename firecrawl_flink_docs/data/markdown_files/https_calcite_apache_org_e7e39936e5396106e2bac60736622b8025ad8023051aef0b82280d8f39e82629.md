The foundation for your next high-performance database.

## Standard SQL

Industry-standard SQL parser, validator and JDBC driver.

[SQL →](https://calcite.apache.org/docs/reference.html)

## Query optimization

Represent your query in relational algebra, transform using
planning rules, and optimize according to a cost model.

[Relational algebra →](https://calcite.apache.org/docs/algebra.html)

## Any data, anywhere

Connect to third-party data sources, browse metadata, and optimize by pushing the computation to the data.

[Adapters →](https://calcite.apache.org/docs/adapter.html)
